# Mr You Barber Shop Website

A professional, multilingual (Arabic, French, English) website for Mr You Barber Shop & Hammam.

## Features
- **Multilingual Support:** Fully translated in Arabic, French, and English.
- **Admin Dashboard:** Manage gallery images and service pictures directly from the site.
- **Responsive Design:** Optimized for mobile, tablet, and desktop.
- **Firebase Integration:** Real-time database for images and secure admin authentication.
- **SEO Optimized:** Meta tags configured for professional sharing on WhatsApp, Instagram, and Facebook.

## Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Backend/Database:** Firebase (Firestore & Auth)
- **Build Tool:** Vite

## How to Deploy to Netlify
1. Export this project to your GitHub.
2. Log in to [Netlify](https://www.netlify.com/).
3. Click "Add new site" > "Import an existing project".
4. Select your GitHub repository.
5. Use the following build settings:
   - **Build Command:** `npm run build`
   - **Publish Directory:** `dist`
6. Click "Deploy site".

## Admin Setup
To access the admin features:
1. Go to the footer and click "Admin Login".
2. Log in with your authorized email.
3. Once logged in, you can add/remove images.
4. **Important:** Click the "SAVE PICTURES" button at the bottom to permanently save your changes.
